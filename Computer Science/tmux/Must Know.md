- command invoker
```cmd+b```

- split vertically 
```cmd+b + " ```
- split horizontally
```cmd+b + %```

- tmux detach
```cmd+b + d ```

- list tmux sessions
```tmux list-sessions```

- reenter tmux session
	- `a` is for attach and `-t` is for target session
```bash
tmux --attach
tmux a -t NUM
```
